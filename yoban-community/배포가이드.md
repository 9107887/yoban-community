# 요반 소통 사이트 배포 가이드

코딩 몰라도 OK! 순서대로 따라하시면 됩니다 😊

---

## 전체 흐름

```
GitHub (코드 저장) → Supabase (데이터베이스) → Vercel (인터넷 배포)
```

---

## STEP 1 — GitHub에 코드 올리기

1. [github.com](https://github.com) 회원가입 (무료)
2. 우측 상단 `+` → **New repository** 클릭
3. Repository name: `yoban-community` 입력 → **Create repository**
4. 이 `yoban-community` 폴더 전체를 GitHub에 업로드
   - 업로드 방법: 생성된 저장소 페이지에서 **"uploading an existing file"** 클릭 후 폴더 드래그앤드롭

---

## STEP 2 — Supabase 세팅 (데이터베이스)

### 2-1. 계정 만들기
1. [supabase.com](https://supabase.com) → **Start your project** (무료)
2. GitHub 계정으로 로그인

### 2-2. 프로젝트 생성
1. **New project** 클릭
2. Name: `yoban-community`
3. Database Password: 안전한 비밀번호 설정 후 **꼭 저장해두기!**
4. Region: `Northeast Asia (Seoul)` 선택
5. **Create new project** 클릭 (1~2분 소요)

### 2-3. 데이터베이스 테이블 생성
1. 왼쪽 메뉴 **SQL Editor** 클릭
2. 이 폴더의 `supabase/schema.sql` 파일을 열어서 전체 내용 복사
3. SQL Editor에 붙여넣고 **RUN** 클릭
4. ✅ "Success" 메시지가 나오면 완료

### 2-4. API 키 확인
1. 왼쪽 메뉴 **Settings** → **API**
2. 다음 두 값을 복사해서 메모장에 저장:
   - **Project URL**: `https://xxxxxxxx.supabase.co`
   - **anon public key**: `eyJhxxxx...` (긴 문자열)

### 2-5. 관리자 계정 생성
1. 왼쪽 메뉴 **Authentication** → **Users**
2. **Add user** → **Create new user** 클릭
3. 이메일과 비밀번호 입력 (이게 관리자 로그인 계정입니다)
4. **Create user** 클릭

---

## STEP 3 — Vercel 배포 (인터넷에 올리기)

### 3-1. 계정 만들기
1. [vercel.com](https://vercel.com) → **Sign Up** → GitHub으로 로그인

### 3-2. 프로젝트 배포
1. **Add New** → **Project**
2. `yoban-community` 저장소 선택 → **Import**
3. Framework Preset: `Next.js` (자동 선택됨)

### 3-3. 환경변수 설정 ⭐ 중요!
**Environment Variables** 섹션에서 아래 두 개 추가:

| Name | Value |
|------|-------|
| `NEXT_PUBLIC_SUPABASE_URL` | STEP 2-4에서 복사한 Project URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | STEP 2-4에서 복사한 anon public key |

### 3-4. 배포
**Deploy** 클릭 → 2~3분 후 🎉 배포 완료!

배포된 URL: `https://yoban-community-xxxx.vercel.app`

---

## STEP 4 — 마무리 설정

### 쇼핑몰 링크 버튼 수정
`app/page.tsx` 파일에서 아래 부분을 실제 쇼핑몰 주소로 변경:
```
href="https://요반쇼핑몰주소.com"
```
→ 실제 쇼핑몰 URL로 교체 후 GitHub에 다시 업로드하면 자동 재배포

### 기존 쇼핑몰에 링크 버튼 추가
쇼핑몰 관리자에서 버튼/배너에 아래 링크 삽입:
```
https://yoban-community-xxxx.vercel.app
```

---

## STEP 5 — 관리자 페이지 사용법

배포된 URL 뒤에 `/admin` 추가:
```
https://yoban-community-xxxx.vercel.app/admin
```

| 탭 | 기능 |
|----|------|
| 📅 메뉴 관리 | 날짜별 메뉴 추가/삭제 |
| 📩 요청 목록 | 고객 메뉴 요청 확인 + 전화번호 보기 |
| 💬 게시판 | 고객 글 확인, 답글 달기, 숨김 처리 |
| 📋 공지 | 요반 업데이트 노트 작성 |
| 🗳️ 투표 | 고객 투표 만들기/종료 |

## 자주 묻는 질문

**Q. 비용이 드나요?**
- GitHub: 무료
- Supabase: 무료 (월 50만 건 이하 무료, 소규모 사이트 충분)
- Vercel: 무료 (상업 사이트는 월 $20 Pro 플랜 권장)

**Q. 도메인을 연결하고 싶어요.**
- Vercel 대시보드 → 프로젝트 → **Settings** → **Domains** → 도메인 추가

**Q. 메뉴를 많이 한번에 추가하고 싶어요.**
- Supabase 대시보드 → **Table Editor** → `menus` 테이블에서 직접 여러 행 입력 가능
